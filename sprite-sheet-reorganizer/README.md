# Sprite Sheet Reorganizer
A simple console app made to help me organize sprites. 

Drop your png into the `input` folder and run the program. 
Tell it what 8 directions your sprite is facing in order, and it will reorder it to the directions factorio wants. 

Adjust sprite size. My sprites are 162x162 for units and 128x128 for turrets. 

It will automatically create a new file called `output` and place the reorganized sprite sheet there.
